# SocialAutoPost Knowledge Wiki

LLM-maintained markdown wiki for durable project knowledge.

Start here:

- `wiki/index.md` - catalog of synthesized pages
- `wiki/log.md` - chronological change log
- `AGENTS.md` - maintenance rules for Codex/LLM agents

Raw sources and stable links live under `raw/`.

The wiki is intended to be edited over time as experiments, tools, errors, and workflows evolve.
